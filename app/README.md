# DF2B - Don't Forget 2 Breathe

Твій кишеньковий простір безпеки. Ментальна підтримка 24/7.

## Функції

- 🆘 **SOS Кнопка** - Миттєва допомога при панічних атаках
- 🤖 **AI-Психолог** - 5 профілів спілкування для різних груп
- 🫁 **Дихальні практики** - Квадратне, 4-7-8, когерентне дихання
- 🎧 **Аудіо-простір** - Безпечні саундскейпи
- 📊 **Трекер настрою** - Щоденний чек-ін та статистика
- 📳 **Тактильна підтримка** - Вібрація синхронізована з диханням

## Технології

- React + TypeScript
- Tailwind CSS
- shadcn/ui
- PWA (Progressive Web App)
- Local Storage для офлайн-роботи

## Встановлення

### Як PWA

1. Відкрий застосунок у Chrome/Safari
2. Натисни "Поділитися" (iOS) або "Встановити" (Android/Desktop)
3. DF2B з'явиться на головному екрані!

### Для розробки

```bash
# Встанови залежності
npm install

# Запусти dev сервер
npm run dev

# Збери для production
npm run build
```

## Налаштування бекенду

1. Створи `.env` файл:
```bash
cp .env.example .env
```

2. Додай URL бекенду:
```
VITE_API_URL=http://localhost:3001
```

## Структура проекту

```
src/
  components/     # UI компоненти
  sections/       # Екрани застосунку
  hooks/          # Кастомні хуки
  data/           # Дані (профілі, вправи)
  types/          # TypeScript типи
public/
  manifest.json   # PWA маніфест
  sw.js           # Service Worker
```
